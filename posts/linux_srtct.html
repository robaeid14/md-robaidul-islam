Tips to Practice

  • Navigate long commands using Ctrl + A, Ctrl + E, Alt + B, Alt + F.
  • Search old commands with Ctrl + R instead of arrows.
  • Pause/Resume jobs with Ctrl + Z, bg, fg.
  • Use Tab to autocomplete everything possible.
  • Clear screen quickly with Ctrl + L.
  • Combine shortcuts to work smarter, e.g., Ctrl + R then Ctrl + A.
